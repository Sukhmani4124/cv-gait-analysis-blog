# Human Gait Analysis using Computer Vision

Welcome to our Computer Vision project blog.

This project focuses on analyzing human walking patterns using classical computer vision techniques. The aim is to study gait movement by extracting motion-based features from walking videos without relying on deep learning models.

Through this project, we explore concepts such as image preprocessing, motion analysis, feature extraction, and pattern analysis to understand how visual data can be used to interpret human movement.
